Design Notes
Logo V1 — Mark + Wordmark
Geometric ensō circle with a 55° gap at ~1 o'clock. Gradient stroke from #5B5BD6 → #9B8CF5 with round linecaps. A small filled terminus dot anchors the arc end — a nod to the calligraphic brushstroke origin of the ensō mark. Wordmark in Onest SemiBold at tight tracking. Primary use: nav, header, email signature.
Logo V2 — Typographic Wordmark
"enso" in Onest ExtraBold (800 weight), tight letter-spacing. A small gradient circle to the right acts as a brand accent — a compressed reference to the ensō mark. Thin separator rule followed by "REVENUE OPERATING SYSTEM" in wide-tracked Onest Medium. Primary use: document headers, pitch decks, proposals.
Color System
Mark gradient: #5B5BD6 → #9B8CF5 (both backgrounds). Dark bg: white wordmark (#FFFFFF). Light bg: near-black wordmark (#09090F). The mark gradient works on both — no separate color variants needed for the icon.
Font: Onest Variable
All text uses Onest, loaded from the local font file at ../assets/Fonts/Onest/. Variable font with weight axis 100–900. Logo V1 uses weight 650. Logo V2 uses weight 800 (wordmark) and 500 (descriptor). Falls back to system sans-serif if font unavailable.
